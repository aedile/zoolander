On Dropdown click, Boostrap immediately closes all menus with `Dropdown.clearMenus()` preventing submenus by default because the parent menus will hide as a result.

For this to work, we will need to patch Bootstrap's Dropdown.toggle method with the following snippet

Third Tier Sub Menus do not work with this code

  
var $parentDropdown = $parent.parents('.dropdown');
if ($parentDropdown.length) {
    $parentDropdown.find(toggle).first().dropdown('toggle');
}
